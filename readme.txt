== Description ==

= About BitCredits.io =

BitCredits.io allows your store to offer bitcoins as a payment option, while never having to hold them yourself. We take care of the legalities related to crypto-currencies.

= How it works =

BitCredits provides an easy-to-use widget that displays on your order confirmation page. Your customers will send a payment to the displayed address before completing the order.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload this entire directory to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Woocommerce/Settings
1. Go to the "checkout" tab
1. Click "Bitcredits"
1. Enter your API Key
1. Click "Save"

== Frequently Asked Questions ==

= Why Bitcoin =

The Bitcoin network provides an easy way to instantly send payments around the world. Those payments are final and irrevocable, they can therefore not be charged back.
